About This Calculator

This is nothing but the simplest of simple calculators, with a few features that I find helpful.

I hope it shows off my JQuery and Bootstrap skills sufficiently -- that it is designed for the smallest of screens (the browser on my LG Phoenix 3 phone) but switches up to a larger screen when the viewport allows. It also uses object-oriented Javascript to run the calculations.

Other features I'm proud of:

  • Mouse or keypad entry! The number and math keys on your desktop keyboard also work for entry.
  • Handles decimals (particularly decimal sums) gracefully -- no more than 3, which is maybe arbitrary, but I'm this for my own use. It will not pad out the sum with zeroes -- 3/2 comes back with 1.5, not 1.500, 4/2 = 2, not 2.00.
  • Divide by zero? Try it, get a little surprise.
  • Used CSS Grid for the layout, for a very minimal HTML footprint.

    • If you'd like to see the code behind this (well-commented if I do say so myself) it's up on GitHub.
